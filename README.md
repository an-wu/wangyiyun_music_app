#  网易云音乐移动端

## 接口文档地址 [网易云音乐 NodeJS 版 API](https://binaryify.github.io/NeteaseCloudMusicApi/#/?id=neteasecloudmusicapi)

## 接口仓库地址  [Binaryify/NeteaseCloudMusicApi](https://github.com/Binaryify/NeteaseCloudMusicApi)

## 完成的功能
+ 首页
+ 歌单页
+ 播放器页
+ 登陆页

`npm i`

`npm i serve`
